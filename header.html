Component

Header

You'll see quick menu and navigating to other pages with ease from here.

Overview

The Header contains two components, quick menu and navigation menu components. The quick menu is in control to show personal or business purpose in navigation menu. Quick menu also shows language selection, contact us link, sign in/sign out button, and search menu.

Header

How to Achieve this Feature/Technical

To achieve all features available in Header, API integration is needed to interact between application with database. The API is responsible to serve data to be used for menu lists on navigation.
