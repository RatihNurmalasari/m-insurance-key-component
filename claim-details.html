Claim Details

Your claim details are presented here.

Overview

After submitting claim number and taxonomy code on previous page, then the expected response have returns from server, this page will open. The user should be able to view member information and related claim details

How to Achieve this Feature/Technical

To achieve all features available in Claim Details, it needs API integration to interact between application with database. API is responsible for handling request by user who have entered their claim number and taxonomy. All APIs return response in JSON format and will be mapped in user interface.
