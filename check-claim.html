Component

Check Claim

Check your claim status and claim details by inputting claim number and taxonomy code.

Overview

The user can fill check claim status form in here by submitting the claim number and choose the taxonomy code that populated from Taxonomy API. If the data that user submitted pass the acceptance criteria, application call Check Claim API and navigate to Claim Details page.

How to Achieve this Feature/Technical

To achieve all features available in Check Claim, it needs API integration to interact between application with database. API is responsible for handling request by user who have entered their claim number and taxonomy. All APIs return response in JSON format and will be mapped in user interface.
